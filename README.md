# Прогнозирование финансовых спредов методом Ньютона–Рафсона

Программный комплекс краткосрочного прогнозирования дневной логарифмической доходности валютных пар (валютного спреда) на основе авторегрессионных моделей **AR(1)** и **ARX(1)**. Ключевая особенность — параметры оцениваются методом **Ньютона–Рафсона** по **аналитически выведенным** градиенту и матрице Гессе, что даёт точную наблюдаемую информационную матрицу Фишера и, как следствие, статистически корректные доверительные интервалы и t-статистики без зависимости от шага конечной разности.

Дипломная работа (ТПУ, прикладная математика и информатика).

## Что внутри

- **Оценка методом Ньютона–Рафсона** с аналитическими производными — квадратичная сходимость за 1–2 итерации.
- **Статистический вывод через матрицу Фишера**: стандартные ошибки, t-статистики, p-значения, 95%-доверительные интервалы.
- **Экзогенная переменная** — индекс волатильности рынка акций как прокси риск-аппетита («индекс страха»): VIX (по опционам на S&P 500) для EUR/USD, RVI (по опционам на индекс РТС) для рублёвых пар. Собственного индекса волатильности у валютной пары не существует, поэтому используется общерыночный индикатор.
- **Бенчмарки**: ARIMA, SARIMAX, нейросеть LSTM.
- **Walk-forward бэктест** с метриками MAE, RMSE, MASE, коэффициента Тейла.
- **Межпарный анализ** четырёх пар: EUR/USD, USD/RUB, EUR/RUB, CNY/RUB.
- **Интерактивное веб-приложение** на Streamlit (графики на Plotly).

## Установка и запуск

Требуется Python 3.11+.

```bash
# 1. виртуальное окружение
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/macOS:
source venv/bin/activate

# 2. зависимости
pip install -r requirements.txt

# 3. пакетный прогон (оценка + прогноз + бэктест)
python main.py

# 4. межпарный анализ
python compare_pairs.py

# 5. интерактивное приложение
streamlit run app.py
```

Источник данных переключается в `config.py` (`DATA_SOURCE = "yahoo"` для EUR/USD, `"moex"` для рублёвых пар). Котировки кэшируются в `data/`, поэтому проект запускается и без доступа к API.

## Метод

Для модели с нормальными инновациями минимизируется отрицательное логарифмическое правдоподобие. Шаг Ньютона–Рафсона:

```
θ_{k+1} = θ_k − H(θ_k)^{-1} · ∇F(θ_k)
```

Градиент `∇F` и матрица Гессе `H` выведены аналитически (для AR(1) — размерности 3 и 3×3, для ARX(1) — 4 и 4×4). В точке сходимости `H` совпадает с наблюдаемой информационной матрицей Фишера, её обращение сразу даёт ковариационную матрицу оценок — отсюда «бесплатные» доверительные интервалы.

## Основные результаты

- На паре EUR/USD четыре модели (AR(1)-НР, ARX(1)-НР, ARIMA, SARIMAX) показывают сопоставимую точность и уверенно обходят наивный прогноз (MASE ≈ 0,54).
- Коэффициент при индексе волатильности формально значим на 5%-уровне только для CNY/RUB (p ≈ 0,02). Однако эффект экономически мал и статистически неустойчив: с поправкой на множественные сравнения он незначим, а на под-выборке 2023–2026 годов исчезает и меняет знак. Значимость на полной выборке порождена турбулентностью 2022 года и структурным сдвигом рынка, а не устойчивой предсказуемостью пары.
- Практическая ценность подхода — не в направленном прогнозе (дневная доходность близка к непредсказуемой), а в корректной количественной оценке его неопределённости.
